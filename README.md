# Harshit Chauhan Portfolio

A Vite + React portfolio site with Tailwind CSS, Motion animations, Lenis smooth scrolling, and an editorial brutalist visual style.

## Run Locally

```powershell
npm install
npm run dev
```

Open the local URL shown in the terminal, usually `http://localhost:3000`.

## Useful Commands

```powershell
npm run build
npm run lint
npm run clean
```
